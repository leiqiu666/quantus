# SDD · 沪深港通资金流向

> **CLI 命令：** `market_hsgt pull-by-date-range`
> **源码入口：** `src/etl/cli.py`
> **Tushare 接口：** [`moneyflow_hsgt`](https://tushare.pro/document/2?doc_id=47)

---

## 1. 概述

按 SSE 开市日逐日调用 Tushare `moneyflow_hsgt`，拉取沪股通/深股通/港股通每日资金流向，upsert 到 PostgreSQL `market_moneyflow_hsgt` 表。每日 1 条记录，采用 **event_driven** 完整性（0 条也视为已拉取）。

> 积分要求 2000+（5000 积分 500 次/分钟）。限流 500/min。

### 触发方式

```bash
uv run ./src/etl/cli.py market_hsgt pull-by-date-range
uv run ./src/etl/cli.py market_hsgt pull-by-date-range --start-date 20141117 --end-date 20251231
uv run ./src/etl/cli.py market_hsgt check-complete
```

### 前置依赖

| 依赖 | 说明 |
|------|------|
| `TUSHARE_API_KEY` | 需 2000+ 积分 |
| `HSGT_FLOW_START_DATE` | floor（legacy env，推荐 `20141117` 沪港通开通日） |
| `stock_trade_calendar`（SSE） | 开市日来源 |
| PostgreSQL | 目标库连接 |

## 3. 分层架构

```
CLI → MoneyflowHsgtStrategy
       ├─ TradeCalStrategy.ensure_trade_cal(SSE)
       ├─ CompletenessEngine.backfill_keys()  ← event_driven
       └─ for trade_date in pending:
            └─ MoneyflowHsgtWorkflow.pull_moneyflow_hsgt_by_date(trade_date)
                 ├─ MoneyflowHsgtExtract → TushareMoneyflowHsgtClient → ts.moneyflow_hsgt(trade_date=)
                 └─ MoneyflowHsgtLoad → bulk_upsert → market_moneyflow_hsgt
```

**源码路径：** `src/etl/{strategy,workflow,extract,load,client}/market/market_moneyflow_hsgt_*` + `src/entities/data_entities/market/market_moneyflow_hsgt_entities.py`

---

## 4. 数据契约

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str(8) | 交易日期 |
| ggt_ss | float | 港股通（上海）百万元 |
| ggt_sz | float | 港股通（深圳）百万元 |
| hgt | float | 沪股通百万元 |
| sgt | float | 深股通百万元 |
| north_money | float | 北向资金百万元 |
| south_money | float | 南向资金百万元 |

**冲突键：** `trade_date`（唯一）

**完整性：** `source_name=market_moneyflow_hsgt`，`event_driven=True`，`period_stock_count=1`

**Admin SSE：** `market_hsgt_check` → `MoneyflowHsgtStrategy.check_complete`

---

## 5. 边界

- 某日 API 返回空：入库 0 条，`mark_date_pulled` 标记已拉取
- `finalize_moneyflow_hsgt` 过滤 `trade_date` 非 8 位记录
