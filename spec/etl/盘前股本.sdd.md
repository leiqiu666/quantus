# SDD · 盘前股本

> **CLI 命令：** `stock_premarket pull-by-date-range`；`stock_premarket check-complete`
> **Admin 大类：** 基础类 / 交易日（`group_id=stock_basic_trade_date`）
> **SSE task_key：** `stock_premarket_check`
> **源码入口：** `src/etl/cli.py`
> **Tushare 接口：** [`stk_premarket`](https://tushare.pro/document/2?doc_id=329)

---

## 1. 概述

按 SSE 开市日逐日调用 Tushare `stk_premarket(trade_date=)` 拉取全市场盘前股本（总股本、流通股本、昨收、涨跌停价），upsert 到 PostgreSQL `stock_premarket` 表。为精确市值、换手分母、涨跌停校验提供独立股本快照。

> 完整性按全市场交易日股票数 95% 阈值守门（与 `market_daily_basic` 同类 by-date 模式）。限流 500/min。

### 触发方式

```bash
uv run ./src/etl/cli.py stock_premarket pull-by-date-range
uv run ./src/etl/cli.py stock_premarket pull-by-date-range --start-date 20200101 --end-date 20251231
uv run ./src/etl/cli.py stock_premarket check-complete
```

### 前置依赖

| 依赖 | 说明 |

## 3. 分层架构

```
CLI → StockPremarketStrategy.pull_premarket_by_date_range(start, end)
       ├─ TradeCalStrategy.ensure_trade_cal()
       ├─ CompletenessEngine.backfill_keys()          ← trade_date × 95%
       └─ for trade_date in pending (tqdm):
            └─ StockPremarketWorkflow.pull_premarket_by_date(trade_date)
                 ├─ StockPremarketExtract → TushareStockPremarketClient
                 │    └─ ts.stk_premarket(trade_date=) → finalize
                 └─ StockPremarketLoad → bulk_upsert_postgresql → stock_premarket
```

## 5. 业务规则

1. **按交易日全市场：** 传 `trade_date` 一次拉取当日全部股票盘前股本。
2. **完整性：** `source_name=stock_premarket`，`date_column=trade_date`，分母为当日交易股票数，阈值 95%。
3. **Upsert 幂等：** `(trade_date, ts_code)` 联合唯一。
