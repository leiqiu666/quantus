# SDD · 指数基本信息

> **CLI 命令：** `index pull-basic-snapshot`
> **Tushare 接口：** [`index_basic`](https://tushare.pro/document/2?doc_id=94)

---

## 1. 概述

按市场（MSCI/CSI/SSE/SZSE/CICC/SW/OTH）全量拉取 Tushare `index_basic`，upsert 到 `index_basic` 表。为指数 universe、基准映射、`index_daily` 代码列表扩展提供基础数据。

拉取模式：**snapshot**（无完整性校验）。

## 2. 分层

```
CLI → IndexBasicStrategy.pull_snapshot
  → IndexBasicWorkflow.pull_index_basic_snapshot
    → IndexBasicExtract → TushareIndexBasicClient
    → IndexBasicLoad → index_basic
```

## 3. 表结构

| 表名 | `index_basic` |
| 冲突键 | `(ts_code)` |

字段：`ts_code,name,fullname,market,publisher,index_type,category,base_date,base_point,list_date,weight_rule,desc,exp_date`

## 4. Admin / SSE

| 项 | 值 |
|----|-----|
| group_id | `index_month` |
| column_key | `index_basic` |
| sse_task_key | `index_basic_pull` |
| 补位 | 全量 snapshot，忽略日期区间 |
