# SDD · 指数日线行情

> **CLI 命令：** `index pull-daily-by-code-range` / `index check-daily-complete`
> **Tushare 接口：** [`index_daily`](https://tushare.pro/document/2?doc_id=95)

---

## 1. 概述

按指数代码区间调用 Tushare `index_daily`，upsert 到 `index_daily` 表。默认遍历 `INDEX_CODES` + 主要基准指数，并尝试从 `index_basic` 加载 CSI/SSE 综合指数扩展列表。

拉取模式：**by-code**；完整性按 **逐指数 × 交易日**（`CompletenessEngine.pull_by_index`，`index_date_unit=day`）。

## 2. 分层

```
CLI → IndexDailyStrategy.pull_index_daily_by_code_range
  → for ts_code in resolve_index_daily_codes():
       IndexDailyWorkflow.pull_index_daily(ts_code, start, end)
         → IndexDailyExtract → TushareIndexDailyClient
         → IndexDailyLoad → index_daily
```

## 3. 表结构

| 表名 | `index_daily` |
| 冲突键 | `(ts_code, trade_date)` |

字段：`ts_code,trade_date,close,open,high,low,pre_close,change,pct_chg,vol,amount`

## 4. 完整性

- `source_name`: `index_daily`
- 分母：每个 SSE 开市日 × 指数个数（`resolve_index_daily_codes()`）
- 补拉：`check_complete_per_index`，按指数聚合缺失交易日区间后调用 `pull_index_daily`

## 5. Admin / SSE

| 项 | 值 |
|----|-----|
| group_id | `index_trade_date` |
| column_key | `index_daily` |
| sse_task_key | `index_daily_check` |
| 默认起点 env | `INDEX_DAILY_START_DATE` |
