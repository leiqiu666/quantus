# SDD · 申万行业成分构成

> **CLI 命令：** `index pull-member-all-snapshot`
> **Tushare 接口：** [`index_member_all`](https://tushare.pro/document/2?doc_id=335)

---

## 1. 概述

按 L1 行业代码分批调用 Tushare `index_member_all(is_new=Y)`，upsert 到 `index_member_all` 表。单次 API 上限 2000 行，默认从 `index_classify` 读取 L1 `industry_code` 逐批拉取。

拉取模式：**snapshot**（无完整性校验）。

## 2. 分层

```
CLI → IndexMemberAllStrategy.pull_snapshot
  → IndexMemberAllWorkflow.pull_index_member_all_snapshot
    → IndexMemberAllExtract（按 L1 分批）
    → IndexMemberAllLoad → index_member_all
```

## 3. 表结构

| 表名 | `index_member_all` |
| 冲突键 | `(ts_code, l1_code, l2_code, l3_code, in_date)` |

字段：`l1_code,l1_name,l2_code,l2_name,l3_code,l3_name,ts_code,name,in_date,out_date,is_new`

**Transform：** `out_date` 的 None/NaN 归一化为 `""`（upsert 冲突键要求）。

## 4. Admin / SSE

| 项 | 值 |
|----|-----|
| group_id | `index_month` |
| column_key | `index_member_all` |
| sse_task_key | `index_member_all_pull` |

| 依赖 | 说明 |
