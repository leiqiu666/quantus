# SDD · 前十大流通股东

> **CLI 命令：** `shareholder pull-floatholders-by-date`
> **交互菜单：** 【股东】前十大流通股东区间增量
> **源码入口：** `src/etl/cli.py`
> **Tushare 接口：** [`top10_floatholders`](https://tushare.pro/document/2?doc_id=62)

---

## 1. 概述

按公告日逐日调用 Tushare `top10_floatholders(ann_date=)` 拉取全市场前十大流通股东，upsert 到 PostgreSQL `financial_top10_floatholders` 表。

> 与 `top10_holders` 相同，采用 **event_driven** 完整性。积分要求 2000+。

### 触发方式

```bash
uv run ./src/etl/cli.py shareholder pull-floatholders-by-date
uv run ./src/etl/cli.py shareholder pull-floatholders-by-date --start-date 20100101 --end-date 20251231
uv run ./src/etl/cli.py shareholder check-floatholders-complete --start-date 20100101
```

### 前置依赖

| 依赖 | 说明 |
|------|------|
| `TUSHARE_API_KEY` | 需 2000+ 积分 |
| `TOP10_FLOATHOLDERS_START_DATE` | floor（推荐 `20100101`） |
| `stock_trade_calendar` | SSE 开市日 |

---

## 2. CLI 入口

| 项 | 值 |
|----|-----|
| Typer 子命令组 | `shareholder`（扩展） |
| 命令名 | `pull-floatholders-by-date` / `check-floatholders-complete` |
| 处理函数 | `Top10FloatholdersStrategy.pull_top10_floatholders_by_date` |

---

## 3. 分层架构

```
CLI → Top10FloatholdersStrategy.pull_top10_floatholders_by_date
       ├─ TradeCalStrategy.ensure_trade_cal()
       ├─ CompletenessEngine.backfill_keys()          ← event_driven × ann_date
       └─ for ann_date in pending (tqdm):
            └─ Top10FloatholdersWorkflow.pull_top10_floatholders_by_ann_date
                 ├─ TushareTop10FloatholdersClient (offset/limit 分页)
                 └─ Top10FloatholdersLoad → financial_top10_floatholders
```
