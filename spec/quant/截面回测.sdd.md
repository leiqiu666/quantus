## 1. 概述

最小闭环：单因子截面排序 → 分 N 组（默认 10）→ 月度（或周度）调仓 → 输出 IC / RankIC / 分组净值 / 多空。

```text
调仓日 T:
  Universe(T) → FactorDataset.read_multi([factor], T) → 排序分组 → 目标权重
  交易约束（停牌/涨跌停）过滤不可买
  [T, T_next) 用后复权日收益结算 → 扣 Cost
落盘: backtest/{strategy}/{run_id}/
```

---

## 2. 复权与收益

| 项 | 约定 |
|----|------|
| 持有期收益 | `ret_t = close_adj[t] / close_adj[t-1] - 1` |
| 因子值 | 已由各源按后复权（hfq）生产；回测不二次复权 |
| 涨跌停 / 停牌 | 用未复权价与 `up_limit`/`down_limit`、`stock_suspend`；不可交易则权重挪到现金或跳过（首版：**跳过该标的，剩余组内等权再归一**） |
| T+1 | 首版按「调仓日收盘成交、次日才计入持仓收益」或「调仓日收盘价成交且当日不计收益」二选一；**定稿：调仓日 T 收盘调仓，收益从 T→T+1 开始计** |

## 3. Universe

文件：`src/research/dataset/universe.py`

```python
class UniverseDataset:
    def all_a(
        self,
        trade_date: str,
        *,
        exclude_st: bool = True,
        min_list_days: int = 60,
        exclude_suspended: bool = True,
    ) -> list[str]:
        """
        该日可投资股票：
        - list_date <= trade_date < delist_date（或 delist 空）
        - 可选排除名称含 ST
        - 上市不足 min_list_days 排除
        - 可选排除全天停牌
        """

    def index_members(self, index_code: str, trade_date: str) -> list[str]:
        """Phase 2 可后置；有 index_weight 时可接。"""
```

生存偏差：必须按历史时点用 `stock_list.list_date` / `delist_date`，禁止用「今日股票池」回灌。

## 5. 引擎与策略接口

```python
class CrossSectionEngine:
    def __init__(
        self,
        strategy: SingleFactorStrategy,
        universe: UniverseDataset,
        cost_model: CostModel,
        rebalance_freq: str = "monthly",  # monthly | weekly
        n_groups: int = 10,
    ) -> None: ...

    def run(self, start: str, end: str) -> BacktestResult: ...


class SingleFactorStrategy:
    """按因子值降序分 n_groups；组内等权；可输出多空 = top - bottom。"""

    def __init__(self, factor_name: str, n_groups: int = 10) -> None: ...

    def target_weights(
        self,
        trade_date: str,
        factor_cs: pl.DataFrame,  # ts_code, value
        universe: list[str],
    ) -> dict[str, dict[str, float]]:
        """
        返回 group_id -> {ts_code: weight}，weight 组和为 1。
        group_id: "G0".. 或 "long"/"short"。
        """
```

### BacktestResult

```python
@dataclass
class BacktestResult:
    daily_returns: pl.DataFrame
    # trade_date, G0..G{n-1}, long_short
    portfolio_history: pl.DataFrame
    # trade_date, group_id, ts_code, weight
    trades: pl.DataFrame
    # trade_date, group_id, ts_code, delta_weight, cost
    ic_series: pl.DataFrame
    # trade_date, ic, rank_ic
    summary: dict
    # sharpe, mdd, annual_return, turnover, ic_mean, icir, ...
```

## 6. 绩效模块

`src/research/performance/`

| 文件 | 职责 |
|------|------|
| `ic.py` | 调仓日因子值 vs 下一持有期收益的 Pearson IC / Spearman RankIC；IR/ICIR |
| `returns.py` | 年化、夏普、最大回撤、卡玛 |
| `report.py` | 写 `report.json` + 可选简单摘要打印 |

前向收益窗口与调仓频率对齐（月调 → 约 20 日或至下一调仓日收益）。
