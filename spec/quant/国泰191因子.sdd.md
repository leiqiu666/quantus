## 1. 约定

| 项 | 值 |
|----|-----|
| 因子名 / 目录 | `gtja_alpha{N}`（N=1..191，无前导零） |
| `factor_meta.source` | `国泰191` |
| `factor_meta.category` | `gtja191` |
| `factor_meta.formula` | [`docs/国泰191.md`](../../docs/国泰191.md) 原文 |
| 复权 | 后复权 OHLC（`*_adj`） |
| VWAP | `(amount * 10) / vol`（amount 千元、vol 手 → 元/股近似） |
| 基准指数 | `000300.SH`（`BANCHMARKINDEXOPEN/CLOSE`，需 `index_daily`） |
| Alpha30 | **仅 meta，不算值**（依赖 MKT/SMB/HML） |
| 存储 | `factor/gtja_alpha{N}/dt=YYYYMM/` |

---

## 2. 计算编排

按月：读日 K（含 lookback≥260 交易日）→ 构建价量面板 → 对目标 Alpha 列表求值 → 裁剪当月 → `FactorParquetLoad`。

单 Alpha 失败：记日志、计入 skip 列表，不中断整批。公式预处理：`?:`→`WHERE`、`=`→`==`、`and/or`→`AND/OR`。
