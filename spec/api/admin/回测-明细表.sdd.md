# SDD · 回测明细表

> **HTTP：** `GET /api/admin/quant/backtest/runs/{run_id}/tables?name=portfolio|trades|returns`  
> **模式：** ① Service 读  
> **源码：** `quant.py` + `BacktestRunQueryService`

---

## 1. 概述

读取某次回测落盘 Parquet，供「投研分析 → 回测明细」ProTable。

Query：`name` 必填，枚举 `portfolio` | `trades` | `returns`。

可选：`trade_date`、`group_id`、`ts_code`、`limit`（默认 5000，最大 20000）。

响应：`{ "name", "columns": [...], "rows": [...], "total": N }`（total 为过滤后行数；超 limit 截断并在前端提示）。

404：run 不存在或对应 parquet 缺失。
