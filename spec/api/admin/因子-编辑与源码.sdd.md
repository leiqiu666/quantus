> **HTTP：**  
> - `GET /api/admin/quant/factor/{factor_name}`  
> - `PUT /api/admin/quant/factor/{factor_name}`  
> - `GET /api/admin/quant/factor/{factor_name}/source`  
> **模式：** ① Service 读 / 轻量写  
> **源码：** `src/api/routers/admin/quant.py`

---

## 1. 详情

`GET /api/admin/quant/factor/{factor_name}`

响应在列表字段基础上增加：`impl_kind`、`python_path`。

---

## 2. 更新

`PUT /api/admin/quant/factor/{factor_name}`

| 字段 | 何时可改 |
|------|----------|
| display_name / category | 均可 |
| formula | 仅 `impl_kind=formula` |
| impl_kind / python_path / factor_name | 不可改 |

国泰：保存的 `formula` 在下次计算时优先于 `docs/国泰191.md`。
