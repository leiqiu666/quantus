# SDD · 调度-任务管理

> **模式：** ① Service 读/写（同步 JSON）；手动触发异步执行  
> **前缀：** `/api/admin/scheduler`  
> **源码：** [`src/api/routers/admin/scheduler.py`](../../../src/api/routers/admin/scheduler.py)

---

## 1. 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/commands` | 25 条命令 + referenced_by |
| GET | `/overview` | 覆盖看板汇总 |
| GET | `/jobs` | 任务列表 |
| POST | `/jobs` | 创建任务 |
| GET | `/jobs/{job_key}` | 任务详情 |
| PATCH | `/jobs/{job_key}` | 更新 |
| DELETE | `/jobs/{job_key}` | 删除 |
| POST | `/jobs/{job_key}/run` | 手动触发（SSE 进度） |
| POST | `/runs` | 执行历史分页 |
| GET | `/runs/{run_id}` | run + steps |

---

## 2. 创建任务请求

```json
{
  "job_key": "pre-market-basic",
  "name": "开盘前基础数据",
  "schedule_kind": "weekdays_at",
  "schedule_time": "09:25",
  "run_on_trading_day": true,
  "enabled": true,
  "command_keys": [
    "trade-cal-pull-history",
    "stock-pull-list-a",
    "suspend-pull-by-date"
  ]
}
```

## 4. 调用链

| 端点 | Service |
|------|---------|
| commands / overview | `ScheduleOverviewService` |
| jobs CRUD | `ScheduleJobService` |
| runs | `ScheduleRunService` |
| 手动 run | `ScheduleRunService.trigger_job_async` → `scheduler.runner` |
