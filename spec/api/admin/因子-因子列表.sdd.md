# SDD · 因子列表查询（分页）

> **HTTP：** `GET /api/admin/quant/factor/list`  
> **响应：** `{ items, total }`  
> **模式：** ① Service 读  
> **源码：** `src/api/routers/admin/quant.py`

---

## Query

| 参数 | 说明 |
|------|------|
| page / page_size | 默认 1 / 20，page_size≤500 |
| source / category / keyword | 可选筛选 |
