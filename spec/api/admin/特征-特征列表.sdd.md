# SDD · 特征列表 / 种子 / 覆盖刷新

> **HTTP：**  
> - `GET /api/admin/quant/feature/list`  
> - `POST /api/admin/quant/feature/seed`  
> - `POST /api/admin/quant/feature/refresh-coverage`  
> - `PUT /api/admin/quant/feature/{feature_id}`  
> **模式：** ① Service 读/轻量写（同步）  
> **源码：** `src/api/routers/admin/quant.py`

---

## 1. 列表

`GET /api/admin/quant/feature/list?page=1&page_size=20&keyword=&feature_kind=&source_kind=&enabled=`

| 参数 | 说明 |
|------|------|
| page | 从 1 起，默认 1 |
| page_size | 默认 20，最大 200 |
| keyword | 匹配 feature_name / display_name |
| feature_kind | `source` / `derived` |
| source_kind | `kline_daily` / `index_daily` / `derived` |
| enabled | `1` / `0` |

响应：

```json
{ "items": [ { "id": 1, "feature_name": "CLOSE", "...": "..." } ], "total": 16 }
```

---

## 2. 种子

`POST /api/admin/quant/feature/seed`  
按国泰 panel 符号 upsert（按 `feature_name`），不覆盖已有 `remark` / 手动改过的 `display_name` 时可选择仅补缺——本期：**整行 upsert 种子字段**，覆盖区间保留已有值。

响应：`{ "upserted": 16 }`

---

## 3. 刷新覆盖

`POST /api/admin/quant/feature/refresh-coverage`  
扫描仓库写回 `start_date` / `end_date`。

响应：`{ "updated": N, "kline_start": "...", "kline_end": "...", "index_start": "...", "index_end": "..." }`

---

## 4. 更新

`PUT /api/admin/quant/feature/{feature_id}`  
可改：`display_name` / `enabled` / `remark` / `formula` / `sort_order`（不可改 `feature_name`）。
