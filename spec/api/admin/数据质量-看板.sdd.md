# SDD · 量化数据源看板

> **模式：** ① Service 读（同步 JSON）  
> **HTTP：** `POST /api/admin/data-source/dashboard`  
> **分组列表：** `GET /api/admin/data-source/groups`  
> **源码：** [`src/api/routers/admin/data_source.py`](../../src/api/routers/admin/data_source.py)

---

## 1. 概述

按 ETL 大类 + 时间维度返回宽表：每行一个 `date_key`，每列一个数据源的数量与完整性指标。

`group_id` 定义见 [`src/service/etl/completeness_dashboard_config.py`](../../src/service/etl/completeness_dashboard_config.py)。

### 触发示例

```bash
curl -X POST http://localhost:8000/api/admin/data-source/dashboard \
  -H "Content-Type: application/json" \
  -d '{"group_id":"market_trade_date","start":"20260629","end":"20260703","page":1,"count":20}'
```

---

## 2. 调用链

| 层级 | 组件 |
|------|------|
| Router | `get_dashboard` |
| Service | `CompletenessDashboardService.get_dashboard` |
| Config | `completeness_dashboard_config.DASHBOARD_GROUPS` |
| Model | `CompletenessSnapshotModel` / `ReportService` / `KlineDailyService` / `StockSuspendModel` |

---

## 3. 请求

| 字段 | 类型 | 说明 |
|------|------|------|
| `group_id` | string | 看板分组 ID |
| `start` | string \| null | 区间起点（YYYYMMDD 或 YYYYMM） |
| `end` | string \| null | 区间终点 |
| `page` | int | 页码，从 1 起 |
| `count` | int | 每页条数，1–500 |

## 5. group_id 清单

| group_id | 行键 | 列 |
|----------|------|-----|
| `financial_report_period` | 报告期 | 三表+指标+预告/快报/审计 |
| `financial_ann_date` | 公告日 | 股东户数/前十大股东 |
| `kline_trade_date` | 交易日 | 日K/复权/涨跌停/技术因子 |
| `market_trade_date` | 交易日 | 每日指标/资金流/两融/北向/大宗/龙虎榜 |
| `index_month` | YYYYMM | 指数权重 |
| `stock_basic_trade_date` | 交易日 | 停复牌条数 |
